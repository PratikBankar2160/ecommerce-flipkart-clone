# Flipkart Clone Ecommerce Website

## Description
A full-stack ecommerce web application inspired by Flipkart.  
Backend: Spring Boot + PostgreSQL + JWT  
Frontend: React + Axios + Bootstrap

## Features
- User registration & login with JWT authentication
- Product listing, search & filter
- Cart management
- Admin panel for product management

## Tech Stack
- Java, Spring Boot, Hibernate, PostgreSQL
- React, Axios, Bootstrap
- JWT for authentication

## Setup
1. Clone the repository
2. Backend: Run Spring Boot application
3. Frontend: `npm install && npm start`
